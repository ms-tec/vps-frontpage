Software

I programmeringstimerne bruger vi primært programmeringssproget Python. Derudover skal du bruge en editor: et program du kan skrive din kode i. Det kan entent være en meget simpel editor, f.eks. Notepad++ til Windows, eller du kan vælge en med mere funktionalitet, f.eks. Visual Studio Code. Det vigtigste er at det er en editor lavet til programmering — Word kan ikke bruges.

  • Python

    Her kan man downloade den nyeste version af Python.

  • Visual Studio Code

    Visual Studio Code er en god editor der virker på alle styresystemer. Husk at installere udvidelsen Python — så kan du køre dine Python-programmer direkte fra VS Code.

  • Notepad++

    Notepad++ er en simpel editor til Windows. Den kan ikke så meget, til gengæld fylder den heller ikke så meget.

  • 7-zip

    I programmering skal du ofte aflevere zip-filer med flere kodefiler. 7-zip er et godt værktøj til at arbejde med zip-filer og andre arkivfiler på Windows.

Biblioteker og dokumentation

På nedenstående links kan du finde information om Python-programmering, om mange af de biblioteker vi bruger i undervisningen, og diverse andre emner.

  • Object-Oriented Programming in Python

    Bog om Python-programmering på engelsk.

  • Sanic dokumentation

    Sanic er et web-framework til Python, som vi bruger i programmeringstimerne når vi arbejder med web-udvikling. På ovenstående link finder du den officielle dokumentation.

  • Dear PyGui dokumentation

    Dear PyGui er et moderne GUI-framework til Python, som vi bruger når vi gerne vil lave desktop-programmer.

  • Jinja

    Jinja er et template-sprog, som kan bruges til at lave HTML-filer med dynamisk indhold.

  • SQLite3 til Python

    SQLite3 er et simpelt fil-baseret database-bibliotek. Når man installerer Python, får man automatisk et Python-bibliotek til SQLite3 med. På ovenstående link findes dokumentation og eksempler.

  • MDN Web Docs

    Mozilla Foundation har lavet nogle virkelig gode sider med dokumentation og tutorials for alt hvad der har med HTML, CSS, og JavaScript at gøre. Dem finder du på ovenstående link.

  • jsfiddle

    På jsfiddle kan man hurtigt og nemt eksperimentere med HTML, CSS, og JavaScript. Nyttigt til at undersøge hvordan man får et lille element på sin hjemmeside til at se rigtigt ud, inden man integrerer det i resten af koden.
